# Installation
* ## Run:
        pip install instagrapi
        pip install praw
        pip install Pillow
* ## Fill out the config.ini file
* ## Run the first_run.py file
* ## Run the main.py file
- - - 
# To Automate the file you can add it to the task scheduler to run upon the boot of the computer
